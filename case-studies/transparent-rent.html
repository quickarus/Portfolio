TransparentRENT Rent Tracker

Project Overview

A complete front-end design and development project for TranspaRENT, an intuitive tool for tracking and analyzing rental market data. The platform helps both renters and landlords make informed decisions about rental properties.

Initial Design

The initial design included a landing page without any apartment listings. The goal of this page was to introduce the platform to new users, and prompt them to sign up or select a city. Above are several iterations of the landing page, which I designed and coded.

User-Centric Homepage

The homepage features a clean, modern design with intuitive navigation. Users can easily search for properties, track price changes, set up alerts, and find ways to save on rent. The interface emphasizes key actions through clear visual hierarchy and accessible design patterns.

Search Results Experience

The search results page combines a powerful filtering system with an interactive map view. Users can easily browse available properties, compare prices, and identify opportunities in their desired neighborhoods. The interface provides clear price change indicators and availability status.

Building Profile

Detailed building profiles provide comprehensive information about properties, including available units, pricing trends, and property features. The layout emphasizes important details while maintaining a clean, scannable interface that helps users make informed decisions.

Unit Details

The unit details page offers in-depth information about specific apartments, including pricing history, square footage, and amenities. Interactive elements like the pricing timeline help users understand market trends and make data-driven decisions about their rental choices.
